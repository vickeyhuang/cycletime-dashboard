# First run CycleTime script to generate a output csv
```
python3 CycleTime_Calculation_Enhanced.py
```

# Then generate a dashboard via a latest output csv by default
```
python3 delivery_dashboard_generator.py
```

# Generate a dashboard via a custom csv
```
python3 delivery_dashboard_generator.py --input my_data.csv
python3 delivery_dashboard_generator.py -i output_files/my_specific_file.csv
```
